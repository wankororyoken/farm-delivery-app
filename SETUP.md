# 農場配達管理アプリ — 設定手順

Google Apps Script を使うので、Google Cloud Console の設定は不要です。

---

## ステップ 1: スプレッドシートを開く

対象の Google スプレッドシートをブラウザで開きます。

---

## ステップ 2: Apps Script エディタを開く

メニュー → **「拡張機能」** → **「Apps Script」**

---

## ステップ 3: ファイルを作成・貼り付け

エディタに以下の4ファイルを作成し、このリポジトリのコードを貼り付けます。

| エディタで作るファイル | リポジトリのファイル |
|---|---|
| `コード.gs`（最初からある） | `コード.gs` |
| `index.html`（＋ボタンで追加） | `index.html` |
| `style.html`（＋ボタンで追加） | `style.html` |
| `script.html`（＋ボタンで追加） | `script.html` |

HTMLファイルは「ファイルを追加」→「HTML」で追加できます。

---

## ステップ 4: シート名を確認

スプレッドシートのタブ名が **「シート1」** 以外の場合は、
`コード.gs` の先頭を書き換えます。

```js
var SHEET_NAME = 'シート1';  // ← ここをタブ名に合わせる
```

---

## ステップ 5: ウェブアプリとしてデプロイ

1. エディタ右上の **「デプロイ」** → **「新しいデプロイ」**
2. 種類: **「ウェブアプリ」**
3. 設定:
   - 説明: `農場配達管理`（任意）
   - 次のユーザーとして実行: **「自分」**
   - アクセスできるユーザー: **「自分のみ」**（または組織内）
4. **「デプロイ」** → Googleアカウントで承認
5. 表示された **ウェブアプリのURL** をコピー

---

## ステップ 6: iPhoneで開く

1. Safari でウェブアプリのURLを開く
2. 共有ボタン →「ホーム画面に追加」でアプリのように使えます

---

## スプレッドシートの列構成

アプリは以下の列順を想定しています（1行目はヘッダー、2行目からデータ）:

| 列 | 内容   |
|----|--------|
| A  | 日付   |
| B  | 品名   |
| C  | 数量   |
| D  | 単価   |
| E  | 金額   |
| F  | 事業   |
| G  | 納品先 |

---

## コードを更新するとき

1. Apps Script エディタでファイルを編集・保存
2. **「デプロイ」** → **「デプロイを管理」** → 鉛筆アイコン
3. バージョン: **「新しいバージョン」** → **「デプロイ」**

※ URLは変わりません。
